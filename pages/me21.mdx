## 聚合硬件总览

| 模块 | 型号 / 规格 | 算力 / 容量 | 备注 |
|------|------------|------------|------|
| **CPU** | ARM Cortex-A78 × 12核 | 2.65 GHz | SoC 集成 |
| **GPU** | Moore Threads M1000 iGPU | 730 MHz | MUSA 后端，vLLM 推理 |
| **NPU** | m1000-npu（SoC 集成） | **50 TOPS** | 非 PCIe 外挂 |
| **内存** | LPDDR 统一内存 | **32 GB** | CPU + GPU + NPU 共享 |
| **存储** | NVMe MAP1202 SSD | **476.9 GB** | 深圳实创忆，122GB已用 |
| **VPU** | 硬件编解码器 | 编/解码各 0% | 视频加速单元 |
| **风扇** | pwm-fan | 5000 rpm | 主动散热 |

> ⚠️ 本设备 NPU 为 **SoC 集成型**（50 TOPS），与 MQ50 的 PCIe 外挂型后摩 XH2A（210 TOPS）**完全不同**，不兼容 HLIELLama。

## NPU 详细信息

| 项目 | 详情 |
|------|------|
| 名称 | m1000-npu（of_node compatible: mthreads,m1000-npu） |
| 架构 | **SoC 片内集成**（非 PCIe 外挂） |
| 平台地址 | `a00000.npu`（SoC 内部地址空间） |
| 内核模块 | `npucore`（基于 Vivante Graphics Driver） |
| 设备节点 | `/dev/npucore` |
| sysfs 路径 | `/sys/devices/platform/a00000.npu` |
| IOMMU 组 | 24 |
| 标称算力 | **50 TOPS** |
| 当前状态 | unsupported（常开，无运行时挂起）|
| 驱动类型 | m1000_npu_model_zoo 兼容（MTNN 格式） |
| 监控命令 | `cat /sys/devices/platform/a00000.npu/power/runtime_status` |

> **与 MQ50（210 TOPS）的核心差异：**
> - MQ50 NPU = 后摩 XH2A IPU，PCIe 4.0 x4 外挂，24GB 专属内存
> - ME21 NPU = 摩尔线程 m1000-npu，SoC 集成，共享系统内存
> - 两者**驱动、格式、SDK 完全不兼容**

## 监控命令参考

```bash
# 整机概览（CPU/GPU/温度/风扇）
mthreads-smi

# CPU + 内存实时监控
htop

# GPU 电源状态
cat /sys/devices/platform/soc@0/800000.gpu/power/runtime_status

# NPU 状态
cat /sys/devices/platform/a00000.npu/power/runtime_status
cat /sys/devices/platform/a00000.npu/of_node/compatible

# 查看运行中进程
ps aux | grep -iE "vllm|qwen|model_manager"

# 模型管理
/opt/mt-ai/llm/model_manager.sh status
/opt/mt-ai/llm/model_manager.sh list

# 磁盘 IO
iostat -x 2
```

## 与 MQ50（AI BOX）硬件对比

| 维度 | ME21（本机，50TOPS） | MQ50（AI BOX，210TOPS）|
|------|---------------------|----------------------|
| SoC 板型 | M1000 ME21 | M1000 MQ50 |
| CPU | Cortex-A78 × 12，2.65 GHz | Cortex-A78 × 12，2.65 GHz |
| 内存 | 32 GB 统一 | 32 GB 统一 |
| 存储 | 476.9 GB NVMe | 953.9 GB NVMe |
| GPU | M1000 iGPU，730 MHz | M1000 iGPU，730 MHz |
| **NPU 品牌** | **摩尔线程 m1000-npu** | **后摩 HoumoNPU LQ50** |
| **NPU 算力** | **50 TOPS** | **210 TOPS** |
| **NPU 接口** | **SoC 集成（a00000）** | **PCIe 4.0 x4 外挂** |
| **NPU 内存** | **共享系统内存** | **24 GB 专属内存** |
| NPU 设备节点 | `/dev/npucore` | `/dev/xh2a_ipu0` + `/dev/npucore` |
| 推理框架 | vLLM + MUSA（GPU） | HLIELLama（NPU）|
| 风扇 | 有（5000 rpm） | 无（被动散热）|
| VPU | 有（视频编解码）| 无 |
| HLIELLama 兼容 | ❌ | ✅ |
| m1000_npu_model_zoo | ✅ | ❌ |
